変数をベタ書きするだけで良い

# 例

```bash
command='ls'

$command
```

# 結果

```bash
a.txt b.txt c.txt
```

# 応用

つなげることも出来る。

```bash
$command /music
```

```
Bomb_A_Head.wav Feeling_good.wav Funky_Gutsman.wav
```


# 環境

- sh 3.2
- bash 3.2
