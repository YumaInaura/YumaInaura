```bash
$ [TEXT] | awk '{ print length, $0 }' | sort -n -s | cut -d" " -f2-
```

と書く。

# 例

```bash
pyramid=$(cat << EOM
AA
A
AAAA
AAAAA
AAA
EOM
)

echo "$piramid" | awk '{ print length, $0 }' | sort -n -s | cut -d" " -f2-
```

# 結果

```
A
AA
AAA
AAAA
AAAAA
```

# 解説

いちど各行頭に数字をつけてからソートして、また数字を消す、ということをやっているっぽい。

# 環境

- bash 3.2
