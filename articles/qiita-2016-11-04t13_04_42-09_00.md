# 手順

## 適当に ruby ファイルを作る

```rb:/tmp/example.rb
class Example
  def self.yes
    puts 'YES!'
  end
end
```

## pry を立ち上げて先程の require する

```bash
$ pry
```

```rb
 [1] pry(main)> require '/tmp/example'
=> true
```

## メソッドを実行する

```rb
[2] pry(main)> Example.yes
YES!
```

# 環境

- Pry version 0.10.4 on Ruby 2.3.1
