---
title: "# jq コマンドで #JSON オブジェクトの配列から正規表現で絞り込みをかける例"
emoji: "🖥"
type: "idea"
topics: ["JSON"]
published: true
---

```
$ echo '[{"name":"Alice", "sex": "woman"},{"name":"さだし", "sex": "man"}]' | jq '[.[] | select(.name | match("^[a-zA-z]+$")) ]'
[
  {
    "name": "Alice",
    "sex": "woman"
  }
]
```
