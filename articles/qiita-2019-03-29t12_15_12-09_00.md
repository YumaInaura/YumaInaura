なんだよこのエラーメッセージ‥と思っていたら、python コマンドを指定せずに実行して、shell として評価されていた、そんなこと、よくありませんか。

```
$ ./hello.py
./hello.py: line 2: syntax error near unexpected token `"Hello py"'
./hello.py: line 2: `print("Hello py")'
```

shebang を書けばもうそんな悲劇とはおさらばです。

```py
#!/usr/bin/env python3

print("Hello py")
```

ほらね。

```
$ chmod +x ./hello.py
```

```
$ ./hello.py  
Hello py
```
