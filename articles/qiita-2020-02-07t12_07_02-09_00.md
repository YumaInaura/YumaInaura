```
result=$(some_command) && echo "$result" | pbcopy && pbpaste
```

# 例

```
$ result=$(ls) && echo "$result" | pbcopy && pbpaste
./
../
a.txt
b.txt
```

コンソールで変数を出力


```
$ echo "$result"
./
../
a.txt
b.txt
```
