* 「ヒアドキュメントモード」は勝手につけた名前。
 
```
$ cat | pbcopy
Alice
Bob
Carol
```

`Ctrl+d` で終了

一度決定した行には戻れないので注意

## 確認

```
$ pbpaste
Alice
Bob
Carol
```


## Version

- bash 4.2
- Mac OSX High Sierra
