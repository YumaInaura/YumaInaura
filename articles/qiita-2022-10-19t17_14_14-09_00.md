```
gem install color
```

```rb
require "color"

Color::CSS[:red].to_rgb.to_a.map { |rate| (rate * 255).round }
# => [255, 0, 0]

Color::CSS[:orange].to_rgb.to_a.map { |rate| (rate * 255).round }
# => [255, 165, 0]
```
