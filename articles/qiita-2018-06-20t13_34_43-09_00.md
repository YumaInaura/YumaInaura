# JSONファイルの例

配列の中にハッシュが連なっている例。

```json
[
  {
    "name": "Alice"
  },
  {
    "name": "Bob"
  },
  {
    "name": "Carol"
  }
]
```

# valueで絞り込む場合

```
cat name.json | jq '.[] | select(.name == "Bob")'

{
  "name": "Bob"
}
```

# valueで除外する場合
 
```
$ cat name.json | jq '.[] | select(.name != "Bob")'

{
  "name": "Alice"
}
{
  "name": "Carol"
}
```
