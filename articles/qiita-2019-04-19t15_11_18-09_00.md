- `\b` が単語境界

# python

```py
>>> import re
>>> re.findall(r'\b\w+\b', 'A whole new-World under_the_sea')
['A', 'whole', 'new', 'World', 'under_the_sea']
```

# ruby

```rb
[4] pry(main)> "A Whole new-world under_the_sea".scan(/\b\w+\b/)
>>> ["A", "Whole", "new", "world", "under_the_sea"]
```
