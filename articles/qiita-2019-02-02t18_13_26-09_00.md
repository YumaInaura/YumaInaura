python で テキストから複数個のハッシュタグを正規表現で抽出して、結果を配列で得る

# Example

```py
import re

match = re.findall(r'(#[^\s]+)', 'I like #cat and #dog both.')

# ['#cat', '#dog']
```

# One liner

```
>>> import re; match = re.findall(r'(#[^\s]+)', 'I like #cat and #dog both.'); match
['#cat', '#dog']
```
