Dockerfileの記述例

```
RUN apt-get update && apt-get install -y --no-install-recommends
```

# 時間対策、余計なもの入れないためのオプションらしい

>デフォルトだと recommends しているだけの必須ではないパッケージも一緒に入って時間がかかるので --no-install-recommends をつけるのが常套手段
