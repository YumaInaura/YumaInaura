- `continue` が効かない
- `next` はなんか違う
- `exit` で次に行けそう

# script

```py
#!/usr/bin/env python3
from IPython.terminal.embed import InteractiveShellEmbed
for i in range(5):
  ipshell = InteractiveShellEmbed()
  ipshell()
```
