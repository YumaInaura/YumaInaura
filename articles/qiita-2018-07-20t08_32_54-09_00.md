`$ bundle gem` で作成されたディレクトリであることが前提。

# rakeの場合

```
$ rake build 
```

- コマンドが簡潔
- pkg ディレクトリにgemがビルドされる ( `.gitignore` されているので git差分を汚さない )

# gem コマンド

```
$ gem build yourgem.gemspec
```

- コマンド引数にgemspecを指定しなければいけない
- カレントディレクトリにgemファイルがビルドされてしまう


# 環境

- Bundler version 1.16.2
- gem 2.7.6
