# 例

普通なら文字列全体が返るところを

```bash
$ echo 'Find me please'
Find me please
```

マッチした文字だけ返してくれる

```bash
$ echo 'Find me please' | grep -o me
me
```

# 環境

- bash 4.4
- Mac OSX
