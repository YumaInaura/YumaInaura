# Controller example

use `:except`
not `:permit`

```rb
class UserController < ApplicationController
  def index
    permitted_params = params.except(:foo).permit(:page)

    ...
  end
end
```

# How to test

Maybe Rspec does not support invalid Get with RequestBody method
