# 問題

ネストしたハッシュの中で配列を扱いたい
なおかつその配列に、いきなり .push とかしたい。

こんな感じで。

```rb
nested = {}

nested[:a][:b].push :c
nested[:a][:b].push :d

puts nested # => {:a=>{:b=>[:c, :d]}}
```

# 解決

```rb
nested = Hash.new do |h1,k1|
  h1[k1] = Hash.new do |h2,k2|
    h2[k2] = []
  end
end
```

```rb
nested[:a][:b].push(:c)
nested[:a][:b].push(:d)

puts nested # => {:a=>{:b=>[:c, :d]}}
```

# 環境

- ruby 2.3.1p112 (2016-04-26 revision 54768) [x86_64-darwin15]
