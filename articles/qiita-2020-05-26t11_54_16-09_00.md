- カッコを使ってキャプチャする
- gsub + ブロック記法で マッチ結果に対して好きなメソッドを実行できる

```rb
"ab cd ef".gsub(/(ab|ef)/) { "#{$1.upcase}" }
# => "AB cd EF"
```

# Bad Case

ブロックでないと、複数個のマッチ結果がある場合に、置換結果が全て同じものになってしまう

```rb
"ab cd ef".gsub(/(ab|ef)/, "#{$1.upcase}")
# "EF cd EF"
```
