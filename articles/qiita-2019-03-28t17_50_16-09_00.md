---
title: #jq コマンドで特定のキーを持つオブジェクトだけを絞り込み検索する例
emoji: "🖥"
type: "idea"
topics: ["jq"]
published: true
---

```shell
$ echo '{"alice":"value"}{"bob":"value"}' | jq 'select(has("bob"))'
{
  "bob": "value"
}
```

```shell
$ echo '[{"alice":"value"},{"bob":"value"}]' | jq '.[]' | jq 'select(has("bob"))'
{
  "bob": "value"
}
```
