# Able to specify multiple parts

複数を出力することもできる

```rb
echo "a,b,c" | awk -F'\,' '{ print $1 $2 }'
ab
```

# comma in awk syntax makes space separated text  in stdout 

awk 記法としてのカンマを使うと、結果も分割出力される

```rb
echo "a,b,c" | awk -F'\,' '{ print $1, $2 }'
a b
```
