# ~/.gitconfig

全部 add して 名前決め打ちで commit するエイリアスを作っておく。

```
[alias]
  append = !git add . && git commit -m 'no named commit'
  ap = !git append
```

# 結果

なんとも快適。

# ひとつ前に戻る

このエイリアスを多用している。

```
[alias]
  break = reset HEAD^
```
