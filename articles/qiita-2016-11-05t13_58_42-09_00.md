# 問題

文字変換中に <kbd>Ctrl + k</kbd> を押しても何も起きない。

# 解決

<kbd>Ctrl + i</kbd> で変換できた。

# 原因

「Windows風のキー操作」を設定していたからだった。

![image](https://qiita-image-store.s3.amazonaws.com/0/89618/de0fef1d-dbce-a7a1-3a14-e5f68e2d3cf5.png)



# 環境

- Mac OS X El Capitan
