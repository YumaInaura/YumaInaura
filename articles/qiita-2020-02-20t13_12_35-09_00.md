Uder#name が nil かどうかを聞くには User#name? メソッドを使えば良いっぽい

```rb
user = User.first

# When presence
user.update!(name: "xxx")
# user.name?
=> true

# When blank
user.update!(name: "")
user.name?
# => true

# When nil
User.first.name?
user.name?
# => false

```
