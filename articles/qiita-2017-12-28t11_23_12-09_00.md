### まとめ

- ウィンドウは順序を持っている
- 「行」が「関数」を使って「ウィンドウ」に問い合わせる。

これがウィンドウ関数の基本である。

# クエリで再現

( mysql  8.0.3 を利用 )

## データの用意

```
CREATE TABLE scores (name varchar(255), sex varchar(255), score int);
```

```
INSERT INTO scores (name, sex, score) VALUES ('Alice', 'female', 90); 
INSERT INTO scores (name, sex, score) VALUES ('Bob',   'male',   100);
INSERT INTO scores (name, sex, score) VALUES ('Carol', 'female', 70);
```

## ORDER

```sql
SELECT
  name,
  score,
  RANK() OVER (
    ORDER BY score DESC
  ) AS ranking
FROM
  scores
ORDER BY
  name ASC;
```

```
+-------+-------+---------+
| name  | score | ranking |
+-------+-------+---------+
| Alice |    90 |       2 |
| Bob   |   100 |       1 |
| Carol |    70 |       3 |
+-------+-------+---------+
```

## PARTITION

```sql
SELECT
  name,
  score,
  sex,
  RANK() OVER (
    PARTITION BY sex
    ORDER BY score DESC
  ) AS ranking
FROM
  scores
ORDER BY
  name ASC;
```

```
+-------+-------+--------+---------+
| name  | score | sex    | ranking |
+-------+-------+--------+---------+
| Alice |    90 | female |       1 |
| Bob   |   100 | male   |       1 |
| Carol |    70 | female |       2 |
+-------+-------+--------+---------+
```

## FRAME

```sql

SELECT
  name,
  score,
  AVG(score) OVER (
    ORDER BY score DESC
    ROWS BETWEEN 1 PRECEDING AND CURRENT ROW
  ) AS ranking
FROM
  scores
ORDER BY
  name ASC;
```

```
+-------+-------+----------+
| name  | score | ranking  |
+-------+-------+----------+
| Alice |    90 |  95.0000 |
| Bob   |   100 | 100.0000 |
| Carol |    70 |  80.0000 |
+-------+-------+----------+
```
