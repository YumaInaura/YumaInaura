nuxtプロジェクトで、デフォルトで生成されたページを、てきとうに別ファイルとして保存する

```
cp pages/index.vue pages/some.vue
```

確認のため、てきとうに表示する文字を書き換えておく

yarn dev でサーバー起動している場合

```
yarn dev
```

ページ名がそのままURLに利用される

http://localhost:3000/some

でページにアクセスできる
