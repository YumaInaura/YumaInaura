# 問題

- プログラミングコードやJSONとかホワイトボードに書くの地獄
- ノートPC使うよりアナログのまま説明できないか？

#JSON

```
{ "a": 1, "b": 2, "c": 3, "d": 4, "e": 5 }
```

# shell+jqでパース

```
echo '{ "a": 1, "b": 2, "c": 3, "d": 4, "e": 5 }' | jq .
{
  "a": 1,
  "b": 2,
  "c": 3,
  "d": 4,
  "e": 5
}
```
