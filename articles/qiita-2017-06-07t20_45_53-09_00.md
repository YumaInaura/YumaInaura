`app.xxx_path` と書けば良い。

# 例

```
$ rails console
```

```rb
[16] pry(main)> app.root_path
=> "/"
[17] pry(main)> app.summaries_path
=> "/summaries"
[18] pry(main)> app.summaries_path(1)
=> "/1/summaries"
```

# 環境
- Rails 5.1
