配列ではなくこんな書き方をするらしい。なにこれと思ったpython初心者だった。


```py
>>> "orange".startswith(("a","b","c"))
False
>>> "banana".startswith(("a","b","c"))
True
>>>
```

```py
>>> "orange".endswith(("a","b","c"))
False
>>> "banana".endswith(("a","b","c"))
True
```

# tuple というものらしい？

```
>>> type(('a', 'b', 'c'))
<class 'tuple'>
```
