## YAML (ブロックスタイル)

なんだか奇妙なシンタックスに見えるが、空のハイフン行 `-` の後に、さらに配列記法を続ければ良い。

```yaml
- 1   # 配列の1個目
- 2   # 配列の2個目
-     # 配列の3個目には別の配列をネスト
  - 3    # ネストした配列内の1個目
  - 4    # ネストした配列内の2個目
  -      # ネストした配列の３個目には、さらに新しい配列をネスト (以下略)
    - 5
    - 6
```


## JSONでの解釈

```json
[
  1, 
  2, 
  [
    3, 
    4, 
    [
      5, 
      6
    ]
  ]
]
```

## YAML (フロースタイル)

結果は最初の例と同等。

```yaml
[1,2, [3,4, [5,6]]]
```

## YAML (ブロックスタイルとフロースタイルの併用)

結果は最初の例と同等。

```yaml
- 1
- 2
- [3,4, [5,6]]
```
