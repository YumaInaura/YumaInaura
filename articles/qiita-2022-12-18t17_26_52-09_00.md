# プロジェクトを作成

```
composer create-project "laravel/laravel=9.*" laravel9-breeze
cd laravel9-breeze
```

# Breezeのインストール

```
composer require laravel/breeze
php artisan breeze:install
```

# マイグレーション

```
php artisan migrate
```

# ユーザーを作成

```
php artisan tinker

> $user = new User;
> $user->name = "ユーザー名";
> $user->email = "メールアドレス";
> $user->password = Hash::make("パスワード");
> $user->save();
```


# サーバー起動

```
php artisan serve
```
