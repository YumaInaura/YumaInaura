# ./version-check.py

```
import sys

print(sys.version_info)

```

# profile

.zsh .bashrc など

```
function docker-python() {
  cat "$1" | docker run -i python
}

```

# exe

```
$ docker-python ./version-check.py
sys.version_info(major=3, minor=7, micro=3, releaselevel='final', serial=0)
```
