ちょっと苦しいがこんな感じ。

```rb
I18n.l Time.zone.now.change(hour: 8), format: '%p%l時'
# => "午前 8時"
```

```rb
I18n.l Time.zone.now.change(hour: 22), format: '%p%l時'
# => "午後10時"
```

# 環境

- i18n (0.8.6)
- rails-i18n (5.0.4)
