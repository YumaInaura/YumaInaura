# 問題

「pages 配下に vue ファイルを置くだけでOK」ではない。

`<NuxtPage />` の記述が必要だ。


# 例

`App.vue`

```vue
<script></script>

<template>
  <div>
    <NuxtPage />
  </div>
</template>
```

`pages/example.vue`

```vue
<template>OK</template>
```
