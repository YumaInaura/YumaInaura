# 例

大文字をメソッドとして定義する。

```rb
def A(arg)
  p arg
end
```

普通にクラスを呼ぶと、クラスとして解釈される。

```rb
A # NameError: uninitialized constant A
```

引数を渡すとメソッドと解釈される。

```rb
A 'mimic!' # "mimic!"
```
