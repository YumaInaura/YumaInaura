# オブジェクト指向で書く

mysqli関数でオブジェクトを作っておいて、それを引き回すっぽい。

## mysqli関数（オブジェクト指向型）

```php
//データベースの接続と選択
$mysqli = new mysqli($host, $username, $password, $dbname);
if ($mysqli->connect_error) {
    error_log($mysqli->connect_error);
    exit;
}
```
