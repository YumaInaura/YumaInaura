というエラーが出た場合。

# 解決

mktemp は `X` をランダムな文字列に置き換えてくれる。

`X` を好きなだけ並べれば良い。

```bash
$ mktemp work.XXXXXX

work_8IQYqz # => 生成されたファイルの名前
```

```bash
$ ls | grep work

work.8IQYqz
```

# 環境

- Mac OSX El Capitan 10.11.4
