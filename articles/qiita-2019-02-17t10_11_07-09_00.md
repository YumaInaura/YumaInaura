# Script

特に意味はないけどpythonで
curl叩くのと変わらない

```py
import requests, os

owner = os.environ.get('OWNER')
repository = os.environ.get('REPOSITORY')
url = 'https://api.github.com/repos/' + owner + '/' + repository + '/issues'

print(url)
res = requests.get(url)

print(res.json())
```
