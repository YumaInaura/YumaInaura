# Solution 

If you enter `/remind list` comment field, a `/remind list` will be displayed, so select "Delete".

![image](https://qiita-image-store.s3.amazonaws.com/0/90607/87919de7-8124-a6e2-1ec2-c630e49092ba.png)

# problem 

I do not know where to cancel the reminder I set once.

- Slack ver 1.1.2 
- Max OSX
