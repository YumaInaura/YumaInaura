.zshrc に以下を追加

```
function peco-history-pbcopy() {
  history -n 1 | tail -r  | awk '!a[$0]++' | peco --layout=bottom-up | tr -d "\r\n" | pbcopy 
}

zle -N peco-history-pbcopy
bindkey '^P^P' peco-history-pbcopy


```

Control + PP で起動
