# クエリの例

サブクエリでいちどIDリストを得てからWHEREに渡すと良さそうだ

```sql
SELECT * FROM examples WHERE id IN (
  SELECT MIN(id) FROM examples GROUP BY name
);

```

# 結果

```
id | name
-- | --
1 | Alice
1 | Bob
1 | Carol
```
