- RUN - イメージ作成時に実行される
- CMD - コンテナ実行時に実行される

# 例

## Dockerfile

```:Dockerfile
FROM alpine

RUN echo "RUN!"
CMD echo "CMD!"
```

## イメージ作成

```
$ docker build . -t example_container
```

```
(色々な処理)

RUN!

(色々な処理)
```

## コンテナ実行

```
$ docker run example_container
```

```
CMD!
```
