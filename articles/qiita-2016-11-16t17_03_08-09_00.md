# 問題

`bundle exec rails server` しようとすると、表題のエラーが起こる。

# 解決

既に立ち上がっている rails のプロセスを殺したら解決した。

```bash
ps ax | grep rails
kill -9 [rails process id]
```

# 環境

- Rails 4.2.6
