Integerという型はないのだ

# Fix

```diff
<template>
  <h1>
    {{ count }}
  </h1>
</template>

<script>
export default {
  props: {
    count: {
-     type: Integer,
+     type: Number,
      default: 1
    }
  }
}
</script>

```
