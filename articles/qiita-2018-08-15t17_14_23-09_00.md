Linuxが良い意味で驚かせてくれた。

## 確認

実行ファイルを作る。

```
touch example.sh
```

コマンド補完で実行しようとするが、なぜか補完できない。なぜだ？

```
./exa
```


## 答え

実行権限をつけていないから。

```
chmod +x example.sh
```

`chmod` すれば補完が有効になる。

```
./example.sh
```

すごいねLinux!

- Docker version 18.03.1-ce, build 9ee9f40
