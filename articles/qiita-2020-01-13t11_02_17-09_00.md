直接指定出来せるっぽい

もっと書き方やり方がありそうだけど

```rb
class RemoveUniqueIndexFromSomeTable < ActiveRecord::Migration[5.2]
  def change
    remove_foreign_key :some_tables, name: "fk_rails_43240587ab"
  end
end
```

db:migrate
