# 例

```rb
require 'active_support/inflector'

module A
  class B
  end
end

p A::B.to_s.demodulize # B
```

# 環境

- ruby 2.4.2p198 (2017-09-14 revision 59899) [x86_64-darwin16]
- activesupport (5.1.5)
