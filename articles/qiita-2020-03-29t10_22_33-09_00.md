- we_xxxxx は Stripe の Web Hook Request の id
- sk_yyyyy は Stripe API 用 の SECRET ( Web hook request の Secret ではない )

```
curl https://api.stripe.com/v1/webhook_endpoints/we_xxxxx -u sk_yyyyy: -d url="https://example.com"
```
