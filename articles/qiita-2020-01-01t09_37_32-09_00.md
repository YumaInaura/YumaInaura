```
[alias]
  checkout-and-say-hello = "!f() { git checkout \"$1\" && echo Hello; }; f"
  checkout-and-say-wow = "!f() { git checkout \"$1\" && echo Wow; }; f"

```

# Run


```

Your branch is up to date with 'origin/master'.
Wow
```

```
$ git checkout-and-say-hello master

Hello
```
