例

```
$ docker-compose ps
                Name                              Command               State                                Ports
----------------------------------------------------------------------------------------------------------------------------------------------
some_app_1           command ...   Up       0.0.0.0:3001->3001/tcp
```

```
docker inspect some_app_1 
```
