# 現象

特殊文字 ㈱ が相互変換できない

```rb

'㈱'.encode("Shift_jis").encode('UTF-8')
# Encoding::UndefinedConversionError: U+3231 from UTF-8 to Shift_JIS

'㈱'.encode("cp932").encode('UTF-8')
# => "㈱"
```
