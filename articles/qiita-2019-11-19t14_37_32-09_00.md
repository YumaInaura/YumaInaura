# Factory_botの設定

`spec/rails_helper.rb`

```rb
RSpec.configure do |config|
  config.include FactoryBot::Syntax::Methods

end
```

これで 

- FactoryBot.create を create と書けるようになる
- FactoryBot.build を build と書けるようになる
- etc etc
