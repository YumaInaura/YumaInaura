# 例

```
[command] $(git diff --name-only HEAD^ | peco)
```

# 具体例

rspec (テスト) を回す場合

```
$ rspec $(git diff --name-only HEAD^ | grep spec/ | peco)
```



# 環境

- git version 2.14.1
- peco version v0.5.1
