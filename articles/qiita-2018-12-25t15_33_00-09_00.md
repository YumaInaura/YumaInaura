# 更新

```
$ brew upgrade ruby-build
```


# リリース版インストール(まだない)

```
$ rbenv install 2.6.0
```

# 開発版インストール

```
$ rbenv install 2.6.0-dev
```


# 一覧

```
bash-4.4$ rbenv install --list | grep 2\.6\.0
  2.6.0-dev
  2.6.0-preview1
  2.6.0-preview2
  2.6.0-preview3
  2.6.0-rc1
```
