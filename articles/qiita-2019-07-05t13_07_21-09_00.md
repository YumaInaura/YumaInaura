title: #ruby で Date = 日付を足し算、引き算して差分の日数を得る
emoji: "🖥"
type: "idea"
topics: ["Ruby"]
published: true
---

例

```rb
(Date.new(2019, 10, 01) - Date.new(2019, 07, 01)).to_i
# => 92
```

くわしく

```rb

Date.new(2019, 10, 01) - Date.new(2019, 07, 01)
# => (92/1)
# 何これ

(Date.new(2019, 10, 01) - Date.new(2019, 07, 01)).class
# => Rational
# Rational というクラスのインスタンスらしい

(Date.new(2019, 10, 01) - Date.new(2019, 07, 01)).to_i
# => 92
# to_i すると差分の日付が得られる
```
