expect を ブロックで囲わなければいけない検証の場合。

```rb
it { expect { some_action }.to change { some_status } }
```

普通に `subject` / `is_expected` を書こうとすると落ちる

```rb
subject { some_action }
it { is_expected.to change { some_status } }
```

ラムダを使えば解決。

```rb
subject { -> { some_action } }
it { is_expected.to change { some_status } }
```


# 環境

- rspec 3.4.0
