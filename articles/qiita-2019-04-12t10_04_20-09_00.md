# JSON input ( eg.json )

```json
[
  {
    "labels": [
      {
        "name": "feature"
      }
    ]
  },
  {
    "labels": [
      {
        "name": "bug"
      },
      {
        "name": "important"
      }
    ]
  },
  {
    "labels": [
      {
        "name": "fix"
      },
      {
        "name": "important"
      }
    ]
  }
]
```


# filter pattern 1

```
$ cat tmp/eg.json | jq '[.[] | select(.labels[] | .name == "fix")]'
```

```json
[
  {
    "labels": [
      {
        "name": "fix"
      },
      {
        "name": "important"
      }
    ]
  }
]
```
