# Ref

[Mac ことえりのキーボードショートカットまとめ（50種類） / Inforati](http://inforati.jp/apple/mac-tips-techniques/system-hints/how-to-use-mac-kotoeri-with-keyboard-shortcut.html)


```
「Control」＋「J」	ひらがなに変換
「Control」＋「K」	カタカナに変換
「Control」＋「L」	全角英字に変換
「Control」＋「;（セミコロン）」	半角カタカナに変換
「Control」＋「:（コロン）」	半角英字に変換
```
