# pages

`pages/users/[id].vue` のようなページを作成する

ディレクトリを切る場合は `pages/users/[id]/index.vue` でも良い

( `_id.vue` ではないので注意 )

# params

useRoute を使う

```js
const route = useRoute()
const id = route.params.id
```
