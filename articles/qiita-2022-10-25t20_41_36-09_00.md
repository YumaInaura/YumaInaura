# TypeError: no implicit conversion of Symbol into Integer

配列に対してシンボルでハッシュを呼ぼうとするとエラー

```rb
[][:a]
# TypeError: no implicit conversion of Symbol into Integer
```

# TypeError: no implicit conversion of String into Integer

配列に対して文字列でハッシュを呼ぼうとするとエラー

```rb
[]["a"]
# TypeError: no implicit conversion of String into Integer
```
