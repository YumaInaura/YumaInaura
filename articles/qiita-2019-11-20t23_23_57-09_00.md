# trait 指定なし

```rb
FactoryGirl.define do
  factory :person do
    name { nil }
    sex { nil }

    to_create { |instance| instance.save(validate: false) }
  end
end
```

# trait 指定あり

こいつで範囲を限定的にできる

```rb
FactoryGirl.define do
  factory :person do
    name { nil }
    sex { nil }

    trait :no_validation do
       to_create { |instance| instance.save(validate: false) }
    end
  end
end
```
