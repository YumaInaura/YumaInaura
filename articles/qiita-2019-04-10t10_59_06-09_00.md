- `.[0:1]` is first element of array
- `.[1:]` is other elements of array exclude first element
- we can concatenate arrays with plus operator `+`

```
$ echo '[1,2,3,4,5]' | jq ".[1:] + .[0:1]"
[
  2,
  3,
  4,
  5,
  1
]
```
