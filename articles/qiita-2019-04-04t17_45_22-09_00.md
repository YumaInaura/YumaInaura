# ./docker-compose.yml

```yml
version: '2'
services:
  db:
    image: mysql:5.7
    volumes:
      - ./db/mysql_data:/var/lib/mysql
    environment:
      - MYSQL_ALLOW_EMPTY_PASSWORD=yes
    ports:
      - "10000:3306"
```

# exe

通常実行

```
$ docker-compose up db
```

バックグラウンド実行はこちら

```
$ docker-compose up --detach db
```
