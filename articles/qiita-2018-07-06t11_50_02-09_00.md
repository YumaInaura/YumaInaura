# 例

```bash
# /bin/bash

echo "arg 1 : $1"
echo "arg 2 : $2"
echo "arg 3 : $3"
```

# 実行例

```bash
$ ./example.sh 1 2 3

arg 1 : 1
arg 2 : 2
arg 3 : 3
```

```bash
$ ./example.sh "1 2 3"

arg 1 : 1 2 3
arg 2 :
arg 3 :
```

# 環境

- GNU bash, version 4.4.0(1)-release (x86_64-apple-darwin15.6.0)
