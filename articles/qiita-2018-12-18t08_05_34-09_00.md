# データの保存はファイルに直接記入でDBは使っていなかった

- データをダイヤモンド `<>` で区切る形式で、当時はよく使われていたような
- ただのテキストファイル
- Shift_JIS で保存する仕様だったので途中でエンコードが大変になった

こんな感じ

```
記事名<>記事番号
投稿者名<>メッセージ<br>メッセージ<br>メッセージ<>投稿時刻
投稿者名<>メッセージ<br>メッセージ<br>メッセージ<>投稿時刻
投稿者名<>メッセージ<br>メッセージ<br>メッセージ<>投稿時刻
```
