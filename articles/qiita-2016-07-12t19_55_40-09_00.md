# こんな感じ

```:june.sh
ruby <<-EOM
  puts 'This is ruby!' 
EOM
```

## 結果

```
$ bash june.sh

This is ruby!
```

# Ruby にかぎらず

何でもできるね。

```:july.sh
perl <<-EOM
  print 'This is perl!!' 
EOM
```

## 結果

```
$ bash july.sh

This is perl!!
```

# 環境

- bash 3.2
