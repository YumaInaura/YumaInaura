`[]` メソッド対して引数を与えれば良い


# Array

```rb
[:a, :b, :c][0]
# => :a

nil[0]
# NoMethodError: undefined method `[]' for nil:NilClass

nil&.[](0)
# nil
```


# Hash

```rb
{a: :x}[:a]
# => :x

nil[:a]
# NoMethodError: undefined method `[]' for nil:NilClass

nil&.[](:a)
# nil
```
