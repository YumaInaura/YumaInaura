One idea. use change syntax and expect "not to change".


# BAD

```rb
it do
  expect { subject }.to change { some }.to(true).and_not_to change {
    some.count
  }.by(0)
end
```

# GOOD

```rb
it do
  expect { subject }.to change { some }.to(true).and change {
    some.count
  }.by(0)
end
```
