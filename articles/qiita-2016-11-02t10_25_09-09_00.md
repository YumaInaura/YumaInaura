# 操作方法

`Ctrl + 0` でサイドバーにフォーカスを移す。

- `上下` : カーソル移動
- `右` : フォルダを開く
- `左` : フォルダを閉じる、または上の階層に移動する
- `Enter` : ファイルを開く

(Sublime歴1年半ではじめて知った)

# 環境

- Mac OSX El Capitan
- Sublimetext Stable Build 3216
