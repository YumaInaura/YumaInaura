```
>>> import urllib.parse
>>> encoded = urllib.parse.quote("<code>a</code>\ndagadga")
>>> encoded
'%3Ccode%3Ea%3C/code%3E%0Adagadga'
>>> decoded = urllib.parse.unquote(encoded)
>>> decoded
'<code>a</code>\ndagadga'
```
