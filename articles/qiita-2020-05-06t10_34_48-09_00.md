# NOTE

[フォーム入力バインディング — Vue.js](https://jp.vuejs.org/v2/guide/forms.html)

# Code

```vue
<template>
  <div>
    <h2>
      {{ message }}
    </h2>
    <input v-model="message">
  </div>
</template>

<script>
export default {
  data () {
    return {
      message: "Please change message"
    }
  },
}
</script>

```
