# 問題

複数行のテキスト

```
A
B
C
```

```
B
C
D
```

を比較して、**前者には存在するが、後者には存在しない値 (A)** を得たい。

# 回答例

```bash:diff.sh
expect=$(cat << EOM
A
B
C
EOM
)

exist=$(cat << EOM
B
C
D
EOM
)

diff=$(join -v 1 <(echo "$expect") <(echo "$exist"))
echo $diff
```

# 結果

```
$ bash diff.sh

A
```

# 環境

- bash 3.2
