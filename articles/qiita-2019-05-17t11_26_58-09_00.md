# script.sh

```
#!/usr/bin/env bash

echo "Hey!"
```

直接実行だと -x オプションをつけられない

```
$ ./script.sh
Hey!

```

直接 bash から実行する

```
$ bash -x /tmp/script.sh
+ echo 'Hey!'
Hey!


```
