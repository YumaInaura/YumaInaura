# settiing

以下をコンソールで直接実行

あるいは

~/.zshrc ~/.bashrc などにエイリアスを登録して関数を作る

```
function touchp() { mkdir -p $(dirname "$1"); touch "$1"; }
```

# execute

```
touchp ~/too/deep/dir/dir/dir/file.txt
```
