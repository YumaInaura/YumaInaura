# しまう

`Command + M`

黄色ボタンを押したのと同じ動作

最初化だと思っていたけど「しまう」と呼ぶらしい

# すべてしまう

Command + Option + H + M

# もとに戻す

嘘だろ、なんだよこの操作。でも出来た。

>「Command + Tabでアプリを選択した後、Optionキーを押してCommandキーを離す」
