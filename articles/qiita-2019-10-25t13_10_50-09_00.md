`hash[:key]` と書くところを

`hash&.[](:key)` と書けば良いっぽい。


```rb
{a: { b: :c } }&.[](:a)&.[](:b)
=> :c

{a: nil }&.[](:a)&.[](:b)
=> nil

nil&.[](:a)&.[](:b)
=> nil
```

わりと分かりづらい。
