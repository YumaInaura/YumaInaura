emoji: "🖥"

---

You can specify both

```rb
class User < ApplicationRecord
  has_one :book
  has_many :books
end
```

userが持つbookが複数ある場合にhas_oneを見ると、最新のbook一個だけが得られた

```rb
user.books.size # 3

user.book # Return latest book instance a user has
```
