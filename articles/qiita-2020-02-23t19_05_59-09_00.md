2020-01-22 -> 2019-12-22 -> 2019-11-22

```js
var x = new Date();

date.setMonth(x.getMonth()-1)

date // 2020-01-22T08:55:29.971Z

date.setMonth(x.getMonth()-1)

date // 2019-12-22T08:55:29.971Z

date.setMonth(x.getMonth()-1)

date // 2019-11-22T08:55:29.971Z

```
