use `$route.query` in view template
use `this.$route.query` in script

# index.vue

```vue
<template>
  <div>
      {{ $route.query }}
  </div>
</template>

<script>
export default {
  created () {
    console.log(this.$route.query)
  },
}
</script>

```

# URL example
http://localhost:3000/url-query?id=1&foo=bar
