# デ

1. デ
2. デ

同じように見えて違うっぽい。なにこれ。

# エディタでカタカナの「デ」が違う文字として認識されている

VsCode の 同じ文字にフォーカスするモードで、選ばれる「デ」と得ればれない「デ」がある

文字検索でも同じく

この記事からコピペして同じ結果になるかどうかは不明。
