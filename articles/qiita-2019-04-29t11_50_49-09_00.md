# pyenv

`$ brew install pyenv`

`$ pyenv install 3.7.3`

`$ eval "$(pyenv init -)"`

必要なら .zshrc .bashrc などに追記

`$ python --version`
Python 3.7.3

# pip

`$ pip install django`

# django

`$ django-admin startproject example`

`$ cd example`

`$ ./manage.py runserver`

# http

http://127.0.0.1:8000/ にアクセス
