- いったん `.[]` で配列を解除しておいてから、指定のキーを記述する例
- もっと良いやり方があるかもしれません
```
$ echo '[{"key1":"value1","key2":"value2"}]' | jq '.[] | .key1 + " " + .key2'
"value1 value2"
```
```
$ echo '[{"key1":"value1","key2":"value2"},{"key1":"value3","key2":"value4"}]' | jq '.[] | .key1 + " "  + .key2'
"value1 value2"
"value3 value4"
```
