# playbook

こんな Playbook がある場合

```yaml:example.yml
- name: install
  yum: name=vim state=latest
  sudo: yes
```

# ansible コマンドに置き換える

こんな感じ。

```
ansible -vvvv example -i example_inventory.txt -m "yum" -a "name=vim state=latest" -s
```

- `-m` に使うモジュールを渡す
- `-a` に「モジュールに与える引数」を渡す
- 他の動作指定はオプションで実現したら良さそう ( 今回だと `sudo: yes` を `-s` オプションに置き換えた )

# 環境

- ansible 1.9.6
