# python

へえ python ってこんなこと出来るんだ。すげー、rubyとは違うな。

```
$ echo "print(1,2,3)\nif True:\n\tprint('its True');" | python3
1 2 3
its True
```

# ruby

る、ruby も出来るじゃないかーーー、知らなかったぜ！

```rb
$ echo "p [:a,:b,:c];  if true; puts 'its true'; end" | ruby
[:a, :b, :c]
its true
```
