# Duration は Duration

```ruby
15.minutes
=> 15 minutes
```

# Duration から Integer への変換

```rb
15.minutes.to_i
=> 900
```

# Integer から Duration の引き算

結果はDuration

```rb
60*60 - 15.minutes
=> -15 minutes、3600 seconds
```

# Integer から Duration の引き算 からの Integerへの変換

あたりまえたいそう‥。

```ruby
(60*60 - 15.minutes).to_i
=> 2700
```
