remote の設定はベーシックな場合を想定。

# 例

```
git reset --hard origin/$(git rev-parse --abbrev-ref HEAD)
```

でOK。

# エイリアス登録の例

ご利用は慎重に。

```bash:~/.gitconfig
[alias]
  current = !git rev-parse --abbrev-ref HEAD
  refresh = !git fetch && git reset --hard origin/$(git current)
```

# 環境

- git version 2.9.0
