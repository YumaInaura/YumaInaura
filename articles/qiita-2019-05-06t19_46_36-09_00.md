bytes オブジェクトを生成できるらしい。

```py
>>> b'\xe3\x81\x82'
b'\xe3\x81\x82'

>>> b'\xe3\x81\x82'.decode('UTF-8')
'あ'

>>> 'あ'.encode('UTF-8')
b'\xe3\x81\x82'


>>> b'A' == b'\x41'
True

>>> b'A' == b'A'
True

>>> type(b'A')
<class 'bytes'>

>>> type('A')
<class 'str'>

```
