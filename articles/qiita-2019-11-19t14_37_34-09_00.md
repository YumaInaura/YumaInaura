# Find

```
 *let\(:(\w+)\) *\{ *(.+?) *\}
```

# Replace

```
$1 = $2
```

# Before

```rb
  let(:user) { create(:user, :man) }
  let(:book) { user.book }
```

# After

```rb
user = create(:user, :man)
book = user.book
```

# 操作の例

コマンドパレットで Replace を選んで、C+Enter で全て置換。
