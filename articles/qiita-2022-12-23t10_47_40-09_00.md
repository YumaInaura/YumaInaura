# 問題

.envで設定しているはずが空文字が返って来る。


```php
> env('AWS_SECRET_ACCESS_KEY')
= ""
```

`php artisan cache:clear` でも解決しない。

# 解決


初歩的なミスだった。
.env ファイルで二重に設定していた。


# .envの例

```

AWS_ACCESS_KEY_ID=xxx
AWS_SECRET_ACCESS_KEY=yyy

...(たくさんの行)...

AWS_ACCESS_KEY_ID=
AWS_SECRET_ACCESS_KEY=
```

# 分かったこと

Laravelの .env は丁寧にAWSの例までデフォルトで用意してくれていた。
AWS推しのフレームワークだった。
