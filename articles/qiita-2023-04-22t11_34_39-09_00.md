# コード例

routeの変更を監視することで実現できるようだ

```vue
<script lang="ts" setup>
const route = useRoute()
watch(
  () => route.query,
  () => {
    console.log(route['query'])
  }
)



function move() {
  navigateTo({
    name: 'example',
    query: { foo: Math.floor(Math.random() * 99999) },
  })
}
</script>

<template>
  <button @click="move">move</button>
</template>

```

# 動作例

ボタンを押した時もブラウザバックした時も同じように検知してくれる
