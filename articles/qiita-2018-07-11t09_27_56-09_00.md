# 問題

`$ pyenv shell [virtualenv_name]` した後に `$ pyenv deactivate` が効かない。

# 解決

`$ pyenv shell system` で `deactivate` 状態にできた。


# 環境

- pyenv-virtualenv 1.1.3 (virtualenv unknown)
- pyenv 1.2.5
