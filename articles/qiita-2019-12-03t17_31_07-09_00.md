# settiing

execute on your console bellow command
OR
register to ~/.zshrc ~/.bashrc 

```
function touchp() { mkdir -p $(dirname "$1"); touch "$1"; }
```

# execute

```
touchp ~/too/deep/dir/dir/dir/file.txt
```
