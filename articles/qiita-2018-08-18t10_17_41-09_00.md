```rb
# Both arrays has :C value
([:A, :B, :C] & [:C, :D, :E]).size > 0
# => true

# Both array has no same value
([:A, :B, :C] & [:D, :E, :F]).size > 0
# => false
```

## Versions

- ruby 2.5.1p57 (2018-03-29 revision 63029) [x86_64-darwin17]
