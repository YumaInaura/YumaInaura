# docker-sync.yml

(他の設定は色々省略)

```
syncs:
 app-sync:
  sync_strategy: "${DOCKER_SYNC_STRATEGY}"
```


# .env ファイルに指定する場合

```
DOCKER_SYNC_STRATEGY=rsync
```

```
docker-sync start
```

# コマンドの環境変数に指定する場合

```
DOCKER_SYNC_STRATEGY=unison docker-sync start
```

```
DOCKER_SYNC_STRATEGY=rsync docker-sync start
```

stopなどの場合も同じ環境変数が必要になるので、わりと面倒
