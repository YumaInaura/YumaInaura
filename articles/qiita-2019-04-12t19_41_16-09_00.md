# python
いちど set=集合体  にしてやると扱いやすそう。良いやり方かどうかは不明。

```py
>>> list(set(["a","b","c","d","e"]) - set(["b","d"]))
['a', 'e', 'c']
```

# ruby
配列のまま引き算できるぜ。
```rb
[1] pry(main)> ["a","b","c","d","e"] - ["b", "d"]
=> ["a", "c", "e"]
```
