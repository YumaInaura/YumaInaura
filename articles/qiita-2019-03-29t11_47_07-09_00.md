# 問題

空配列がそのまま標準出力されてしまう。

```
$ echo '[]' | jq '.'
[]
```

# 解決

```
$ echo '[]' | jq 'select(length > 0)'
```

配列の中身があっても大丈夫

```
$ echo '[1,2,3]' | jq 'select(length > 0)'
[
  1,
  2,
  3
]
```

# ダメな例

以下の書き方だと中身がある場合に配列が崩れてしまう。

```
$ echo '[1,2,3]' | jq '.[]'
1
2
3
```
