# negative

` | not` を使う

```
$ echo '[{"count":1},{"count":3},{"count":2}]' | jq '.[] | select(.count==2 | not)'
{
  "count": 1
}
{
  "count": 3
}

```

# positive 
```
$ echo '[{"count":1},{"count":3},{"count":2}]' | jq '.[] | select(.count==2)'
{
  "count": 2
}
```
