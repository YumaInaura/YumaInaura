特にプラグイン不要で利用できるようだ

# コード例

```js
// 定義
const fooCookie = useCookie('foo-key', { secure: true, httpOnly: true })

// セット
fooCookie.value = "bar-value"

// 利用(取得)
fooCookie.value

// 削除
fooCookie.value = null
```
