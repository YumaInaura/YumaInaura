# 答

こんな感じ。

```rb
variable.to_s.empty?
```

# 文字列の場合

```rb
''.to_s.empty? # => true

nil.to_s.empty? # => true
```

# 配列の場合

```rb
nil.to_a.empty? # => true

[].to_a.empty? # => true
```

# ハッシュの場合

```rb
nil.to_h.empty? # => true

{}.to_h.empty? # => true
```
