8桁の場合

```rb
sprintf('%08d', rand(99999999))
=> "15419306"
sprintf('%08d', rand(99999999))
=> "06627530"
sprintf('%08d', rand(99999999))
=> "75424850"
```

別解

```rb
8.times.map { rand(9) }.join
=> "08628150"
8.times.map { rand(9) }.join
=> "88810560"
8.times.map { rand(9) }.join
=> "31853611"

```
