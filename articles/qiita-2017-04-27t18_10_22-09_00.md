オプション付けるだけでOK。

```
gore -autoimport
```
# 結果

`:import time` とか打たなくても、すぐパッケージを使える。

```
gore> time.Parse("2006-01-02", "2017-04-02")
2017-04-02 00:00:00 UTC
nil
```

# 環境

- gore version 0.2.6
- go version go1.8.1 darwin/amd64
