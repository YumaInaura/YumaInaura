```py
{**{"a" : 1}, **{"b" : 2}}
# {'a': 1, 'b': 2}
```

```py
{**{"a" : 1}, **{"a" : 2}}
# {'a': 2}
```

```py
 {**{"a" : 1}, **{"b" : 2}, **{"b" : 3, "c" : 4 }}
# {'a': 1, 'b': 3, 'c': 4}
```
