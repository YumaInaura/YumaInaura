## сЃфсЃбсЃ╝сЃѕсЂФсЃљсЃЃсѓ»сѓбсЃЃсЃЌсЂЎсѓІ

сЃЇсЃЃсЃѕсЂФсЃљсЃЃсѓ»сѓбсЃЃсЃЌсѓњсЂесЂБсЂдсЂісЂЈсЂ«сЂїсђЂСИђуЋфт«Ѕт┐ЃсЂІсѓѓсЂЌсѓїсЂфсЂёсђѓ

сЂдсЂЇсЂесЂєсЂфсЃќсЃЕсЃ│сЃЂтљЇсѓњСйюсЂБсЂдсЂісЂёсЂд `git push` сЂДсЃЇсЃЃсЃѕсЂФсѓбсЃЃсЃЌсЃГсЃ╝сЃЅсЂЎсѓїсЂ░УЅ»сЂёсђѓ

`git push` сѓ│сЃъсЃ│сЃЅсѓѓтѕЦсЂФсЃљсЃЃсѓ»сѓбсЃЃсЃЌућесЂўсѓЃсЂфсЂёсЂЉсѓїсЂЕсђЂсЃљсЃЃсѓ»сѓбсЃЃсЃЌућесЂесЂЌсЂдсѓѓСй┐сЂєсЂЊсЂесЂасЂБсЂдсЂДсЂЇсѓІсђѓ

СЙІ

сЂдсЂЇсЂесЂєсЂфтљЇтЅЇсЂДсЃќсЃЕсЃ│сЃЂсѓњСйюсЂБсЂдсЂісЂЈ

```
$ git branch сЃљсЃЃсѓ»сѓбсЃЃсЃЌућесЂ«сЃќсЃЕсЃ│сЃЂсЂ«тљЇтЅЇ
```

сЃљсЃЃсѓ»сѓбсЃЃсЃЌућесЂ«сЃќсЃЕсЃ│сЃЂсЂФуД╗тІЋсЂЎсѓІ

```
$ git checkout сЃљсЃЃсѓ»сѓбсЃЃсЃЌућесЂ«сЃќсЃЕсЃ│сЃЂсЂ«тљЇтЅЇ
```

сЃфсЃбсЃ╝сЃѕсЂФсЃЌсЃЃсѓисЃЦсЂЎсѓІ

```
$ git push origin сЃљсЃЃсѓ»сѓбсЃЃсЃЌућесЂ«сЃќсЃЕсЃ│сЃЂсЂ«тљЇтЅЇ
```

сѓѓсЂесЂ«сЃќсЃЕсЃ│сЃЂсЂФТѕ╗сѓІ

```
$ git checkout тЁЃсЂ«сЃќсЃЕсЃ│сЃЂсЂ«тљЇтЅЇ
```

## сЃќсЃЕсЃ│сЃЂсЂасЂЉсѓњсЃљсЃЃсѓ»сѓбсЃЃсЃЌсЂЎсѓІ

С╗ісЂёсѓІсЃќсЃЕсЃ│сЃЂсѓњсЃљсЃЃсѓ»сѓбсЃЃсЃЌсЂЎсѓІта┤тљѕ

СЙІ

```
$ git branch сЃљсЃЃсѓ»сѓбсЃЃсЃЌућесЂ«сЃќсЃЕсЃ│сЃЂсЂ«тљЇтЅЇ
```

`git branch` сѓ│сЃъсЃ│сЃЅсЂ»С╗ісЂ«сЃќсЃЕсЃ│сЃЂсЂ«сѓ│сЃћсЃ╝сѓњсђЂтѕЦсЂ«тљЇтЅЇсЂДСйюсЂБсЂдсЂЈсѓїсѓІсђѓ(сЂасЂёсЂЪсЂёсЂЮсѓЊсЂфТёЪсЂў)

ТюгтйЊсЂ»сЃљсЃЃсѓ»сѓбсЃЃсЃЌсЂ«сЂЪсѓЂсЂ«сѓ│сЃъсЃ│сЃЅсЂўсѓЃсЂфсЂёсЂЉсЂЕсђЂсЃљсЃЃсѓ»сѓбсЃЃсЃЌсЂ«сЂЪсѓЂсЂФСй┐сЂєсЂЊсЂесѓѓтЄ║ТЮЦсѓІсђѓ

сЃљсЃЃсѓ»сѓбсЃЃсЃЌсЂФТѕ╗сѓІТЎѓсЂ»сЂЊсЂєсђѓ

```
$ git checkout сЃљсЃЃсѓ»сѓбсЃЃсЃЌућесЂ«сЃќсЃЕсЃ│сЃЂсЂ«тљЇтЅЇ
```

сЂЪсЂасЂЊсЂ«сѓёсѓіТќ╣сЂасЂесђЂУЄфтѕєсЂ«сЃЉсѓйсѓ│сЃ│сЂДgitсЂ«сЃЄсѓБсЃгсѓ»сЃѕсЃфУЄфСйЊсѓњтЅіжЎцсЂЌсЂдсЂЌсЂЙсЂБсЂЪта┤тљѕсЂ»сђЂсѓѓсЂесЂФТѕ╗сЂЏсЂфсЂёсЂ«сЂДТ│еТёЈсЂасђѓ
