# テキスト

```
I
AM
THE
GOOD
VIMMER
```

## スタート

```
echo "I\nAM\nTHE\nGOOD\nVIMMER" > study.txt && vim study.txt
```

## 練習A

1行目から2行目を同時にコメントアウトせよ。

## コマンド

1. `Control + v`
1. `j`
1. `Shift + i`
1. `#`
1. `esc`

( 終わったら `u` で元に戻す )

## 結果

```bash
#I
#AM
THE
GOOD
VIMMER
```

### ステップ

連続して3回繰り返せ。 (自分の決めた回数でも良い)

### ステップ

30秒以内に、連続して3回繰り返せ。 (自分の決めた秒数、回数でも良い)

## 練習B

全ての行を同時にコメントアウトせよ。

1. `Control + v`
1. `Control + d`
1. `Shift + i`
1. `#`
1. `esc`

## 結果

```bash
#I
#AM
#THE
#GOOD
#VIMMER
```

### ステップ

連続して3回繰り返せ。 (自分の決めた回数でも良い)

### ステップ

30秒以内に、連続して3回繰り返せ。 (自分の決めた秒数、回数でも良い)

## 練習C

練習Aと練習Bを順番におこなえ。 (逆順でも良い)

### ステップ

- 練習A
- 練習B
- 練習A

の順番で、あわせて30秒以内におこなえ。 (自分の決めた順番、秒数でも良い)
