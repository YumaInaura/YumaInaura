## 例

```
go env GOPATH
/Users/yuma/go
```

## 悪い例

```
echo $GOPATH

```

```
printenv GOPATH

```

環境変数で GOPATH を設定しておらず、デフォルトが使われている場合は何も分からない。

- go version go1.10.3 darwin/amd64
