```
vim some_file -c 'startinsert'
```

## `-c` オプションって何？

ファイルをロードした後に実行するコマンドを決められる。

```
$ vim --help | grep '\-c '
   -c <command>		Execute <command> after loading the first file
```

( `first file` ってなんだろう )
