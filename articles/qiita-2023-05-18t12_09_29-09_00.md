# 例

```js
ar = [{key: 'foo', enable: false},{key: 'bar', enable: false}]

elm = ar.find(element => element.key === 'foo')
// { key: 'foo', enable: false }

elm.enable = !elm.enable

ar // [ { key: 'foo', enable: true }, { key: 'bar', enable: false } ]
```
