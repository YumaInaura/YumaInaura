`-x` オプションを使う。

# ファイル

```bash:good_bye.sh
echo ABC
```

# 実行

```bash
$ sh -x good_bye.sh
```

# 結果

コマンド自体も標準出力されているのが分かるだろう。

```bash
+ echo ABC
ABC
```

# 環境

- sh-3.2
- bash-3.2
- Max OS X El Capitan 10.11.4
