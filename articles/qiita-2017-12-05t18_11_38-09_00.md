# 手順

## 適当なファイルを用意する

```sql:sample.sql
SELECT
 1 AS A,
 2 AS B,
 3 AS C
```

## mysql コマンドにファイルの中身を渡す

```
mysql -e "$(cat sample.sql)"
```

# 結果

```
+---+---+---+
| A | B | C |
+---+---+---+
| 1 | 2 | 3 |
+---+---+---+
```
