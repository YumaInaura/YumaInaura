小文字でも環境変数に出来る。

```bash:snake.sh
export snake_case='shaaaa!!!!!!!'

printenv snake_case # => shaaaa!!!!!!!
```

逆に大文字でも、普通の変数にできる。

```bash:upper.sh
UPPERCASE='KNOCK DOWN!!!'

echo $UPPERCASE # => KNOCK DOWN!!!
printenv UPPERCASE # => 出力なし
```


# でもなるべく

- 定数は大文字
- 変数は小文字

で書くようにしよう。

# 環境

- bash 3.2
