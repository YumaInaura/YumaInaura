## マイグレーションファイルの例

```rb
  class AddUsersComments < ActiveRecord::Migration[5.1]
  def up
    change_column :users, :first_name, :string, comment: "This is first name of user"
  end

  def down
    change_column :users, :first_name, :string, comment: nil
  end
end
```

## 環境

- activerecord (5.1.6)
- Rails 5.1.6
