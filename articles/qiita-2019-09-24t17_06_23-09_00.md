# vim をインストールしてみる

`apk add vim`

```
bash-4.4# apk add vim
(1/2) Installing lua5.2-libs (5.2.4-r4)
(2/2) Installing vim (8.0.1359-r2)
Executing busybox-1.27.2-r11.trigger
OK: 636 MiB in 140 packages
```

いけた。
