# コマンド

`git reflog` から取ってみる。

```bash
$ git checkout $(git reflog | grep 'checkout: moving' | awk '{ print $8}' | awk '!a[$0]++' | peco)
```

# エイリアス登録

`$ git jump` とかで起動できるようにしておくと便利かもしれない。

```
[alias]
  jump = !git checkout $(git reflog | grep 'checkout: moving' | awk '{ print $8}' | awk '!a[$0]++' | peco)
```

# 環境

- git version 2.14.1
- peco version v0.5.1
- Mac OSX Sierra
