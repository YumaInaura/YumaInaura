# 解決

Preference > Keys 
( もしくは Preference > Profiles > Keys )

の設定を変える。

- ⌥← に `Send Escape Sequence` = `b` を設定
- ⌥→ に `Send Escape Sequence` = `f` を設定

# 補足

- 同じ要領で他のキーにも適用できる。

# 環境

- iTerm2 3.0.12
- Mac OSX

# 3.3.4 のバージョンアップで使えなくなっているっぽい？

3.3.5 Beta では動いた
