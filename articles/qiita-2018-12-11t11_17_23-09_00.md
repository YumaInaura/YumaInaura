#ポイント

− 実はonとかatとか書かなくて良い。
- YYYY-MM-DD 慣れ親しんだ年月日フォーマットでOK
- チャンネル・日付は順序を入れ替えられる
 
#after

```
/remind #channel-name 2018-12-08 11:00 メッセージ
```

#before

```

/remind #channel-name on 12/08/2018 at 11:00 メッセージ
```
