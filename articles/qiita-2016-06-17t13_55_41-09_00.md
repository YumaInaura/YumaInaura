# 検証コード

```bash
man='CATCH ME' # => 1. function の外で変数を定義しても

function jail() {
  echo $man # => 1. 変数は侵入する
  echo $1
  hope='CAN' # => 2. functionの中で変数を定義しても
}

jail 'IF YOU'

echo $hope # => 2. 変数は脱出する

echo $1 # => 3. こいつは脱走できない
```

# 結論

この監獄は脆い。

# 変数閉じ込める方法

local を宣言すること。

```bash
hope='NOT FOUND'

function jail() {
  local hope='FOUND'
}

echo $hope # => NOT FOUND
```

# 環境

- sh 3.2
