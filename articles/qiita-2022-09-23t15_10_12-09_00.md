# コード

可変の配列にはベクタ型というものを使うようだ。

```rs
fn main() {

let mut i = 1;
let mut numbers = vec![];

    loop {
        i += 1;
        let mut prime = true;

        for number in &numbers {
            if i % number == 0 {
                prime = false;
            }
        }
        if prime {
            println!("{i} Prime");
        }

        numbers.push(i);
    }
}
```
