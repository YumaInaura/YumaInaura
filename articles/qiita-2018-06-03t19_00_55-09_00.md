# 答え

インスタンス変数。
メソッドの戻り値とかじゃないので注意。


# コード例

```rb
class A
  def initialize
    @some_instance_variable = :example
  end
end

p A.new # #<A:0x00007fc3930961c8 @some_instance_variable=:example>
```

# ちなみに

- pの結果にはinspectメソッドが使われる。

# 環境

- ruby 2.5.1p57 (2018-03-29 revision 63029) [x86_64-darwin17]
