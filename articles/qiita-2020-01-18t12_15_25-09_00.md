```rb
require 'Faker'
# => false

Faker::Internet::mac_address
# => "ca:ea:fb:ab:cc:b9"
Faker::Internet::mac_address.upcase
# => "32:05:2B:AE:CB:59"
```

ruby的に頑張らなくても良かったんだ

```rb
Array.new(6) { Faker::Number.unique.hexadecimal(digits: 2).to_s.upcase }.join(':')
# => "1E:42:12:27:E2:CB"
```
