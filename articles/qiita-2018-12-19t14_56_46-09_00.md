# 例

```
ssh 127.0.0.1 echo OK! OK!
```

# 引数1個バージョン

```
ssh 127.0.0.1 "echo OK OK"
```

ちょっと分かりにくいがシェル的に、以下どちらの書き方でも動く

# 引数複数バージョン

```
ssh 127.0.0.1 "echo" "OK" "OK"
```
