# ruby redcarpet

```
$ redcarpet  ./tmp/markdown
```

single quote HTML escaped

```html
<h1>header</h1>

<p><em>ã‚</em></p>

<h2>header2</h2>

<ul>
<li>list</li>
<li>list</li>
</ul>

<p><code>
print(&#39;ok&#39;)
print(&#39;yes&#39;)
</code></p>
```

```
$ redcarpet --parse=fenced_code_blocks ./tmp/markdown
```
