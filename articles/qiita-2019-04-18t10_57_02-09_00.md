# ref

[Qiita API v2 documentation - Qiita:Developer](https://qiita.com/api/v2/docs#tag)

# exe

```
for page in {1..100}; do curl -s "https://qiita.com/api/v2/tags?page=${page}&per_page=100&sort=count" | jq '.[].id'; done | tee qiita-tag.txt
```

# caful

途中から limit に引っかかるっぽいので sleep するなりして回避してみてください

```
$ curl -s "https://qiita.com/api/v2/tags?page=1&per_page=100&sort=count"
{"message":"Rate limit exceeded","type":"rate_limit_exceeded"}%
```
