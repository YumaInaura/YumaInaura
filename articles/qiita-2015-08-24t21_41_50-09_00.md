title: "ruby 配列はいちどに初期化しない"
emoji: "🖥"
type: "idea"
topics: ["Ruby"]
published: true
---


```rb
array1 = array2 = []

array2.push('apple')
array2.push('banana')

puts array1
```

って書くと、array2 に push しているはずなのに、

```
apple
banana
```

って結果になってしまう。
array1 = array2 = [] の時点で同じ参照になってしまうぞ。

注意。
