# 例
```rs
let mut numbers = vec![1, 2, 2, 2, 2, 1, 1, 2, 2, 1];

numbers.dedup(); // [1, 2, 1, 2, 1]
```

sortを組み合わせるとユニークに出来る

```rs
let mut numbers = vec![1, 2, 2, 2, 2, 1, 1, 2, 2, 1];

numbers.sort();
numbers.dedup(); // [1, 2]
```
