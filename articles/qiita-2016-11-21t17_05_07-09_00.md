# 手順

まずは全てのコンテナを殺す。

```
docker rm `docker ps -a -q`
```

次に全てのイメージを殺す。

```
docker rmi `docker images -a -q`
```

依存関係によっては消えないものもあるので、あとは手動で殺す。

# 参考

- https://github.com/docker/docker/issues/1329

# 環境

- Docker for Mac
- Docker version 1.12.3, build 6b644ec
