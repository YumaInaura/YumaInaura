#code

```rb
gsub(/[[:space:]]/, '')
```

#pry

```rb
[2] pry(main)> ' ←半角　←全角'.gsub(/[[:space:]]/, '')
=> "←半角←全角"
```

#ref

- `[\p{Z}\t\r\n\v\f]` と同義っぽい
- `\s` では全角スペースが削除されなかった
