# rake 

lib/tasks/some_job.rake

```rb
desc <<~DESC
  This job
  is so
  Great job
DESC

task some_job: :environment do
  # do something
end
```

# describe

descript オプションで全行表示される

```
bin/rails --describe some_job

rails some_job
    This job
    is so
    Great job
```

# tasks

一覧だと一行だけ表示される

```
bin/rails --tasks

...
rails some_job                           # This job
...
```
