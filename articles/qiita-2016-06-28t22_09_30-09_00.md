正確にいうと、解凍先のファイルを作らずに中身だけ見る方法。

# 例

```
$ gunzip -c the_heart_of_pretty_woman.gz
```

普通に中身が表示される。

```
I HATE YOU!
BECAUSE
YOU ARE PEEPING TOM
```

# 応用

圧縮ファイルの行数を数えたり。
色々と使えそう。

```
$ gunzip -c heart_of_pretty_woman.gz | wc -l

```

```
       3
```
