title: "#Ruby で配列の添字・順序を文字列から探す ( find_index )"
emoji: "🖥"
type: "tech"
topics: ["Ruby"]
published: true
---

```rb
["A","B","C","C","D"].find_index("A")
# => 0

["A","B","C","C","D"].find_index("B")
# => 1

# 複数見つかる場合も最初のものが返る
["A","B","C","C","D"].find_index("C")
# => 2

["A","B","C","C","D"].find_index("D")
# => 4

["A","B","C","C","D"].find_index("E")
# => nil
```
