# コード例

`e.target.textContent` が要素のテキストだ
次の例でいうと `Click me` が得られる

```tsx

const Example = () => {

  const handleOnclick = (e: any) => {
    console.log(e.target.textContent )
  }

  return (
    <>
      <div>
        <button onClick={handleOnclick}>Click me</button>
      </div>
    </>
  );
}
export default Example;
```
