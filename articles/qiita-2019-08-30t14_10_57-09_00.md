#Ruby for beginners | Playing a method chain combining class methods and instance methods


次のクラスを使って OK!  と表示させてください

```rb
class Story
 def initialize(to: false)
   @to = to
 end

 def be
   @be = true
   self
 end

 def continued
   puts 'OK!' if @to && @be
 end

 def self.to
   new(to: true)
 end
end
```
