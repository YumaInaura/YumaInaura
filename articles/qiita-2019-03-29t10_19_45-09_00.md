---
title: #jq コマンドで要素全体を配列で囲う、もしくは逆に解除する例
emoji: "🖥"
type: "idea"
topics: ["jq"]
published: true
---


```sh
$ echo '{ "alice": "liddel", "bob": "jobs" }' | jq '[.]'
[
  {
    "alice": "liddel",
    "bob": "jobs"
  }
]
```

```sh
$ echo '{ "alice": "liddel", "bob": "jobs" }' | jq '[.]' | jq '.[]'
{
  "alice": "liddel",
  "bob": "jobs"
}
```
