```go
// https://golang.org/pkg/math/rand/

package main

import (
	"fmt"
	"math/rand"
	"time"
)

func main() {
	// NG
	seed_result := rand.Seed(time.Now().UnixNano())

	// OK
	// Seed does not need return value
	// rand.Seed(time.Now().UnixNano())

	fmt.Println(rand.Int())
}
```

Seed生成した値を使い回すのではなく、一回だけ設定すれば良いみたいだ。なので帰り値は要らない。
