- python3 
- datetime オブジェクトを pytz で localize する

```python
from pytz import timezone
from datetime import datetime

timezone('Asia/Tokyo').localize(datetime.now()).strftime('%Y-%m-%d %H:%M:%S %Z')

# '2019-04-02 19:50:55 JST'
```
