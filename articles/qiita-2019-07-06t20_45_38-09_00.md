```rb
[3,1,2].sort.reverse
=> [3, 2, 1]
```

```rb
[{count: 3},{count: 1},{count: 2}].sort_by { |h| h[:count] }.reverse
# => [{:count=>3}, {:count=>2}, {:count=>1}]
```

別のやり方

```rb
[3,1,2].sort { |a, b| b <=> a }
# => [3, 2, 1]
```

```rb
[{count: 3},{count: 1},{count: 2}].sort_by { |h| -h[:count] }
# => [{:count=>3}, {:count=>2}, {:count=>1}]
```
