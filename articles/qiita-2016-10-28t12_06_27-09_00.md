# 確認

```rb
class Example
  local = nil
  @instance = nil
  @@class = nil

  puts local_variables
  puts instance_variables
  puts class_variables
end
```

## 結果

```
local
@instance
@@class
```

# ちなみに

ローカル変数は宣言より前でも取れる。

```rb
class Example
  puts local_variables
  puts instance_variables
  puts class_variables

  local = nil
  @instance = nil
  @@class = nil
end
```

## 結果

```
local
```

# 環境

- Ruby 2.3.1
