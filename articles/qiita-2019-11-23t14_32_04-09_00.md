全体を env? で囲っておけば、そもそもその環境でしか task一覧に出てこないで良さげ
task  の 中で raise させたりするより良さそう
ただもっと良い書き方があるのかもしれないが

```rb
if Rails.env.development?
  namespace :development do
    namespace :some do
      task run: :environment do
      end
  end
end
```
