# ruby

- File を ERB.new に渡して 変数を binding する
- 引数も渡さずに変数が反映されているのが不思議だが、bindingってそういうことらしい

```rb
require 'erb'

message = 'hello!'
name = 'Alice'

erb = File.read('some.erb')
# => "<%= message %>\n<%= name %>\n"

ERB.new(erb).result(binding)
# => "hello!\nAlice\n"
```
