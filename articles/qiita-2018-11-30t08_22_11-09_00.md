# Youtube

- コンテンツ、情報配信サービス
- ソーシャル機能はあるがソーシャル性はあまりない
- 競合は Twitter Facebook など

# tiktok

- 動画撮影 + ゲームセンター
- 完全にソーシャル命
- 競合はカラオケやゲームセンターなどのアミューズメント

# tiktokはゲームセンター！

tiktokにある機能

- 音に合わせて変顔や笑顔を撮る = 音ゲーだよね！
- 他の人が撮った動画と、画面を半々に分けてコラボ動画が撮れる = ネット機能のあるカラオケだよね！
- スティッカー、スタンプで画面を漏れる = プリクラだよね！
