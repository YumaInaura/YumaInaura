# flat array

```
$ echo '["a","b","c","d","e"]' | jq '. | length'
5
```

# nested array

```
$ echo '[["a","b","c","d"],["e"]]' | jq '. | length'
2
$ echo '[["a","b","c","d"],["e"]]' | jq '.[] | length'
4
1
```
