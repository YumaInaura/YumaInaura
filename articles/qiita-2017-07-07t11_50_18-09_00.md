`$((変数+変数))` みたいな記法を使う。

# 足し算

```bash
$ a=1; b=3; c=5; echo $((a+b+c));
9
```

# 引き算

```bash
$ a=1; b=3; c=5; echo $((a-b-c));
-7
```

# 掛け算

```bash
$ a=1; b=3; c=5; echo $((a*b*c));
15
```

# 割り算

小数点は出ないっぽい。

```bash
$ a=1; b=3; c=5; echo $((c/b));
1
```

# 環境

- GNU bash, version 4.4.0(1)-release (x86_64-apple-darwin15.6.0)
