`boolean` の値をそのまま `when` に渡すだけで良い。
否定でマッチさせる場合は `not` を付ける。

ちなみに `is true` / `is false` などとは書かない。


# playbook の例

```yaml
- hosts: localhost
  tasks:
    - set_fact:
        true_variable: true
        false_variable: false

    - debug:
        msg: "when true"
      when: true_variable
    
    - debug:
        msg: "when not false"
      when: not false_variable
```


# 実行例

```bash
$ ansible-playbook -i ~/development playbooks/when.yml

TASK [set_fact] *********************************************************************************************************************************************************************************************************
ok: [localhost]

TASK [debug] ************************************************************************************************************************************************************************************************************
ok: [localhost] => {
    "msg": "when true"
}

TASK [debug] ************************************************************************************************************************************************************************************************************
ok: [localhost] => {
    "msg": "when not false"
}


```

# 環境

- ansible 2.6.0
