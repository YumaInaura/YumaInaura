docker id には大文字が使えないっぽい。

# Capital letter case (Error)

```
$ docker login
```

```
Username: YumaInaura
Password:
```

```
Error response from daemon: Get https://registry-1.docker.io/v2/: unauthorized: incorrect username or password
```

# Lower letter case (Succeed)

```
$ docker login
```

```
Username: yumainaura
Password:
```

```
Login Succeeded
```
