```py
>>> [h for h in [{"key":"value1"},{"key":"value2"}] if h['key']=='value1' ]
[{'key': 'value1'}]
```

ruby でいうところの select

```rb
[6] pry(main)> [{"key" => "value1"}, {"key" => "value2"}].select { |hash| hash['key'] == 'value2' }
=> [{"key"=>"value2"}]
```
