# つ、ついにできたぞ！

結果としては、ものすごく簡単だった。

# ~/.zshrc

```

function peco-history-current-pbcopy() {
  print "$BUFFER" | tr -d "\r\n" | pbcopy
}

zle -N peco-history-current-pbcopy
bindkey '^P^O' peco-history-current-pbcopy


```

# 実行

- コマンド入力中に Control + P O を押す
