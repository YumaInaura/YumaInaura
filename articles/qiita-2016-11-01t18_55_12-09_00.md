# こうだ。

```rb
[{a: 1, b: 2, c: 3}.to_a.sample].to_h # => => {:b=>2}
```

# ちなみに

これは動かない。

```rb
 {a: 1, b: 2, c: 3}.to_a.sample.to_h
# => TypeError: wrong element type Symbol at 0 (expected array)
```

# 環境

- ruby 2.3.1
